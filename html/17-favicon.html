Belajar Favicon

  • Saat kita membuka web seperti google, facenuk, biasanya pada bagian tab web browser terdapat logo perusahaannya.
  • Bagaiamana cara membuatnya? itu adalah fitur Favicon di HTML
  • Untuk membuat favicon, kita bisa menggunakan tag link pada bagian head
  • dalam link kita perlu menggunakan atribute rel="icon" dan hjref menuju lokasi gambar icon
  • gambar icon bisa file berupa ico, png, jpg, svg, atau gif
