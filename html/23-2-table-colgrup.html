Belajar Membuat Tabel

Tag untuk membuat tabel:
  • Tag table, digunakan untuk membuat tabel
  • Tag tr (table row) untuk membuat row/baris
  • Tag th (table head) untuk membuat kolom header tabel
  • Tag td untuk membuat kolom isi tabel
  • Tag caption untuk keterangan tabel
  • Tag thead untuk gurp header tabel
  • Tag tbody untuk grup isi table
  • tag tfoot untuk grup footer tabel
