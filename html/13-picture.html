Belajar Picture

  • Keuntungan menggunakan picture adalah, kita bisa menggunakan beberapa lokasi gambar dan bisa diatur sesuai dengan ukuran layar. Misalnya, ketika di laya besar, kita bisa kirim gambar dengan ukuran besar, dan ketika di layar kecil, kita bisa kirim gambar yang ukuran kecil.
  • tag picture bukan void element
  • di dalam tag picture, kita bisa menambahkan tag source yang berisi lokasi gambar, dan tag img sebagai default gambar ketika semua kondisi tag source tidak terpenuhi.
  • atribut media di tag source berisikan media query CSS
  • detailnya akan dibahas nanti di kelas CSS
